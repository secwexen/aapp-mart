# Roadmap

The development of **AAPP-MART** is structured into strategic phases:

## Phase 1 – Research & Architecture

- Define system objectives and requirements
- Design core architecture
- Establish attack graph modeling approach
- Develop initial risk scoring methodology

## Phase 2 – Core Implementation

- Implement AAPP (AI Attack Path Predictor)
- Implement MART (Multi-Agent Red Team)
- Integrate deterministic simulation engine
- Develop CLI interface
- Establish input/output contracts and reporting formats

## Phase 3 – Ecosystem & Advanced Features

- Optional REST API for programmatic integration
- Offline / air-gapped operation support
- Containerized deployment (Docker)
- Advanced risk scoring and CVSS integration
- Explainability & audit trail improvements
- Integration with external SOC / SIEM workflows
