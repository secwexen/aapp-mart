# Privacy Policy

**Last updated:** 2026-03-05

AAPP-MART respects your privacy. This project is designed for **authorized security research and enterprise security teams**. It does not collect personal data for marketing purposes.

## 1. Information We Collect

- **Usage Data:** Anonymous analytics to understand interactions with documentation, examples, or optional interfaces.
- **Session Information:** Temporary data to support simulations, API requests, and workflow operations.
- **No Personal Data:** AAPP-MART does not collect or store personally identifiable information (PII) unless explicitly added by users for testing in their authorized environments.

## 2. How We Use Information

- To improve the functionality of AAPP-MART  
- To monitor and enhance simulation performance  
- To provide analytics on usage patterns (fully anonymized)  

## 3. Data Sharing

- AAPP-MART **does not share any data with third parties**.  
- All usage data remains local to the user environment or within authorized enterprise deployments.  

## 4. Data Security

- Data collected for simulations is stored temporarily during runtime.  
- Users are responsible for protecting any sensitive test data in their own environment.  

## 5. Changes to This Privacy Policy

We may update this Privacy Policy as the project evolves.  
All updates will be reflected on this page.  

For information about cookies used in AAPP-MART, see [Cookie Policy](CookiePolicy.md).
