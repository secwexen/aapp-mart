# Attack Path Prediction Engine (AAPP)

## Overview

The Attack Path Prediction Engine (AAPP) is the predictive intelligence component of AAPP-MART. It analyzes assets, identities, privileges, vulnerabilities, trust relationships, and security controls to forecast how an adversary is most likely to move through an environment.

Rather than identifying vulnerabilities in isolation, AAPP evaluates the relationships between security findings and models realistic attack progression paths.
